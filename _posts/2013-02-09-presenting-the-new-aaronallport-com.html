--- layout: post title: Presenting the new aaronallport.com categories: - personal - responsive design - theming - wordpress tags: - design status: publish type: post published: true meta: _edit_last: '1' _wpas_done_all: '1' --- After a few weeks of development, I present the new aaronallport.com Developed as a mobile-first, responsive WordPress theme, I have tried to put more on an emphasis on content, deliberately opting for a stripped-back, useable look. The end result means few menu items, and less clicks to get to each part of the site. I've been able to use SASS in anger, along with coding native JavaScript again (no jQuery to be found here). This has been awesome as the JavaScript is actually the smallest site artefact to load! I intend on putting the theme (with a basic SCSS and JavaScript library on top) up on GitHub in order to give back to the community a little. More on that in a couple of weeks. If there any teething issues found with the site then please drop me an email or send me a message on twitter.
