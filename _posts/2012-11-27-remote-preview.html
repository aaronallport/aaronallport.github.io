--- layout: post title: Remote Preview categories: - external link - responsive design - testing tags: - useful - UX status: publish type: post published: true meta: _edit_last: '1' --- When working on a responsive design, the ability to test on as many hardware platforms (am I'm talking real hardware, not emulators) is critical. There are so many subtle quirks in the sheer amount of available screen resolutions, platforms, and browser support for emerging technologies. Adobe released Shadow, or rather Adobe Edge Inspect a while ago, and it has been factored into the Creative Cloud offering. Trouble is, the last time I used it, it was only supporting iOS and Android. Enter Remote Preview Don't get me wrong, Remote Preview isn't without the odd bug or two, but it sure looks promising as a low-cost, low-effort way to get a device lab up and running. I'll be testing this myself soon, but if anyone has in the meantime, please feel free to share your experiences.
